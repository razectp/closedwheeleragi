# Contributing to ClosedWheelerAGI

First off, thank you for considering contributing to ClosedWheelerAGI! It's people like you that make it a great tool.

## Code of Conduct

Help us keep ClosedWheelerAGI a welcoming and inclusive project. Be kind and respectful to others.

## How Can I Contribute?

### Reporting Bugs
If you find a bug, please create an issue on GitHub. Include:
- A clear, descriptive title.
- Steps to reproduce the bug.
- Your OS and version.
- Any relevant logs from `.agi/logs/`.

### Suggesting Enhancements
We love new ideas! If you have a suggestion:
- Check if it's already been suggested.
- Open an issue describing the feature and why it would be useful.

### Pull Requests
- Follow the existing code style (Go standard formatting).
- Ensure all tests pass (`go test ./...`).
- Provide a clear description of the change.

## Project Philosophy
"Code with Purpose, Build with Pride" - We prioritize quality, security, and context efficiency.
